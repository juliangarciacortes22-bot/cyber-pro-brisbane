Course Overview

Kick start your career in Information and Communications Technology (ICT) with this entry-level qualification. Gain hands-on skills for the role you want, with both a foundation stream and a range of specialised streams.

According to the Australian Government’s Job Outlook service, the number of ICT support technicians is expected to continue growing strongly. This large occupation is likely to see around 62,000 job openings in the next five years.

Generalist IT Support Stream

This stream prepares you to work as an ICT Help Desk Operator. You will learn to care for computer hardware, provide ICT advice to clients, and perform basic system administration support.

Foundation Stream

This stream provides a broad IT foundation, including units from networking, programming, web design, cyber security and cloud. It prepares students for all diploma-level areas.

Cyber Security Awareness

Learn how to identify security threats, protect digital devices, secure information assets, and administer system security. Ideal for entry-level cyber roles.

Networking Stream

Gain the skills to configure computer networks, diagnose and resolve network problems, and install/manage network protocols.

Cyber and Networking Stream

This stream combines fundamental cyber security and networking knowledge, preparing students for deeper study in cloud, cyber security or network engineering, or for entry-level IT roles.

Web Development Stream

Learn to build and maintain websites, use social media tools, create web-ready images, and develop and review simple websites.

Mobile iOS App Development Stream

Learn to build dynamic applications for iPhone and iPad using Swift and Xcode. Understand UI design, app architecture, and how to create user and technical documentation.
